# Gallery Images

This folder contains images for the college gallery section.

## Required Images

Place the following images in this folder:

1. **campus-1.jpg** - Main campus building view
2. **campus-2.jpg** - Academic building/facilities
3. **awareness-rally-1.jpg** - Students in voter awareness rally
4. **awareness-rally-2.jpg** - Community engagement activities
5. **awareness-rally-3.jpg** - Student social awareness programs
6. **event-1.jpg** - College events and activities

## Image Guidelines

- **Format**: JPG or PNG
- **Recommended Size**: 1200x800px (3:2 aspect ratio)
- **File Size**: Keep under 500KB for optimal website performance
- **Naming**: Use lowercase with hyphens (e.g., campus-1.jpg)

## How to Add Images

1. Save your images with the exact filenames listed above
2. Place them in this `/public/gallery/` folder
3. The gallery component will automatically display them
4. Images will be responsive and optimized for all devices

## Current Status

The gallery is set up and ready. Simply add your images to this folder with the correct filenames.
